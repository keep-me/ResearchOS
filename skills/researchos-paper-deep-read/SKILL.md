# ResearchOS Paper Deep Read

先确认目标论文 `paper_id`。如果当前会话已挂载论文，默认优先分析 primary paper。

推荐流程：

1. 调用 `get_paper_detail` 获取标题、摘要和本地 PDF 状态。
2. 检查是否已有精读或三轮分析结果。
3. 如果已有结果足够，直接整理并回答。
4. 如果信息不足，调用 `deep_read_paper`。

回答时重点展开：

- 方法设计与关键模块
- 训练或推理流程
- 实验设置与指标
- 论文局限与改进方向

不要把工具调用过程伪装成已经人工通读全文。
